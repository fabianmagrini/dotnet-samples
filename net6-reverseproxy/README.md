# Reverse Proxy with YARP

Reverse Proxy with YARP and Asp.Net Core 6.0.

References:

* <https://microsoft.github.io/reverse-proxy/articles/getting-started.html>

## Prerequisites

* .Net 6.0
* Visual Studio Code
* Postman

## Create a new project

This is a helloworld api and not for production!

```sh
dotnet new web -n ReverseProxy -f net6.0
dotnet run --project ReverseProxy
```
